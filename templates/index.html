About MeTTa Reasoning

This financial assistant uses MeTTa (Meta Talk Type) for symbolic reasoning, rather than vector databases. This allows for dynamic inference and logical deduction.

Try asking about:

  • ✅ "What happens if inflation increases?"
  • ✅ "What are the effects of high interest rates?"
  • ✅ "How does a recession affect employment?"
  • ✅ "What's the difference between stocks and crypto?"
  • ✅ "What investments are good for a conservative investor?"

MeTTa allows for causal reasoning, so your assistant can explain the relationships between financial concepts!
